<h2>📱 Project Overview</h2>

<p>

Mahoba Insight PWA is a modern Progressive Web Application that transforms the Mahoba Insight website into a native-like news application.

The application fetches all content directly from the existing WordPress backend using the REST API and JWT Authentication without requiring another CMS or database.

The goal is to deliver an experience similar to Reuters, Apple News, and Google News while maintaining the identity of Mahoba Insight.

</p>
